# VendorCollector — Changelog

## 1.0.0
- Initial release
- Detects uncollected mounts, transmog, toys, heirlooms, and pets at vendors
- Shows all uncollected items across every vendor page in a single panel
- Ensemble/set piece detection — flags items that are part of a set you haven't completed
- Currency display for gold, tokens, and vendor-specific currencies
- Settings panel with auto-open and instant-buy options
- ElvUI skin support
